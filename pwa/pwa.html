Add to Home Screen

cruzgodar.com is a progressive web app. That means it can be installed on your device and run like a normal application — it's like setting a bookmark, but better. The app saves your settings even when you close it, and it cuts down on data usage by intelligently caching what it fetches, never downloading the same thing twice. Because of this, it can even be used offline. If you like the sound of all this, you can install the app by following the instructions below that apply to you.


iOS: in Safari, tap the share icon, then select Add to Home Screen from the list of actions.


Android: in Chrome, Firefox, Opera, and other browers that support it, select Add to Home Screen from the menu next to the address bar.


Desktop: In Chrome, select Install Cruz Godar from the menu next to the address bar.
