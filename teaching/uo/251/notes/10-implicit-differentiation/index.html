Section 10: Implicit Differentiation

We typically define functions explicitly. All that means is that $ is written as a function of $, so that we have an equation of the form = f(x)$. The plot of this function is all of the $-pairs that satisfy the equation, and since $ is on its own, we can just plug in a bunch of $-values to $ and get the $-values directly.

It’s possible to write down functions without forcing $ to be alone on one side, though. For example, the equation ^2 + y^2 = 1$ defines a curve: its points are given by all the $ and $ with ^2 + y^2 = 1$. Now it’s not possible to solve for $ explicitly this way, since we’d have to take a square root, which results in = \pm \sqrt{1 - x^2}$. Visually, this means the curve doesn’t pass the vertical line test, which means it’s not a function.

We can still talk about slopes of tangent lines, though. Pictured above is a graph of ^2 + y^2 = 1$, along with two different tangent lines that share the same $-value. When a curve is defined with the $ and $ intermixed and not in the form = f(x)$, we say the curve is defined implicitly, and the process of finding the slopes of tangent lines is called implicit differentiation.

The process takes a second to get used to, because it typically involves heavy use of the chain rule in situations where it might not seem to apply at first glance.

Theorem: implicit differentiation

To find the derivative of a function defined implicitly in terms of $ and $:

1. Take $ rac{d}{dx}$ of both sides. While derivatives of $ work normally, $ is a function of $, so it’s necessary to use the chain rule. We have $ rac{d}{dx}[y] = rac{dy}{dx}$, so for example, $ rac{d}{dx}[ an(y)] = \sec^2(y) \cdot rac{dy}{dx}$.

2. Move all terms containing $ rac{dy}{dx}$ to one side of the equation and all the other terms to the other side.

3. Factor out $ rac{dy}{dx}$ and divide by whatever is left over.

Logarithmic Differentiation

Implicit differentiation unlocks one extra technique: sometimes, difficult functions to differentiate become much easier when we take a log on both sides. Specifically, functions involving strange exponents will turn into multiplication, so we can use the product rule. So: if you have a function of the form = (f(x))^{g(x)}$, start by rewriting it as

19865egin{align*}\ln(y) = \ln((f(x))^{g(x)}) = g(x)\ln(f(x)),nd{align*}19865

and then differentiate implicitly.

Example: logarithmic differentiation

Find $\displaystyle rac{d}{dx}[x^x]$.

Since $ is in both the base and the exponent, we can’t use either the derivative rules for ^b$ or ^x$. Instead, we write

19865egin{align*}\ln(y) = \ln(x^x) = x\ln(x),nd{align*}19865

and then differentiate implicitly to get

19865egin{align*} rac{1}{y} \cdot rac{dy}{dx} &= \ln(x) + x \cdot rac{1}{x}\[4px] rac{dy}{dx} &= y(\ln(x) + 1)\[4px] rac{dy}{dx} &= x^x(\ln(x) + 1).nd{align*}19865
