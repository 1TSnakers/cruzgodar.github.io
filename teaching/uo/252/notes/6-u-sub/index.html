Section 6: hBcSubstitution

At the moment, the only reasonable way we can calculate definite integrals is with the fundamental theorem, which requires taking an antiderivative. In calculus I, there were some derivatives we had to memorize, like $ rac{d}{dx}[\sin(x)] = \cos(x)$, and a few rules like the product and chain rules that let us differentiate more complicated functions. We already have the first part done for antiderivatives, since we can just run all the regular derivatives backward: for example,

19829egin{align*}\int \cos(x)\,dx = \sin(x) + C.nd{align*}19829

What we have yet to develop are the rules analogous to the ones for derivatives. In this section, we’ll start with the simplest and arguably the most useful: the opposite of the chain rule.

Theorem: hBcsubstitution

Given an integral of the form $\displaystyle \int f(g(x))g'(x)\,dx$,

1. Set = g(x)$.

2. Write $ rac{du}{dx} = g'(x)$ and multiply both sides by $ to get = g'(x) dx$.

3. Rewrite the integral as $\int f(u)\,du$.

4. Integrate as usual to get (u) + C$.

5. Back-substitute for $ to get (g(x)) + C$.

This really is just the chain rule backward: since ' = f$, we have $ rac{d}{dx}[F(g(x))] = f(g(x))g'(x)$. In practice, $-sub lets us integrate a composition, as long as the derivative of the inside function is present as a factor in the integral.

Example: hBcsub

Find $\displaystyle \int 6x(3x^2 + 4)^4\,dx$.

We could foil out ^4$, but that’s much more work than is necessary. Instead, we can think of it as a composition of two functions, where the outside function is ^4$ and the inside one is x^2 + 4$. Since

19829egin{align*} rac{d}{dx} \left[ 3x^2 + 4 ight] = 6x,nd{align*}19829

we set = 3x^2 + 4$ and = 6x\,dx$. Then we have

19829egin{align*}\int 6x(3x^2 + 4)^4\,dx &= \int u^4\,du + C\[4px]&= rac{u^5}{5} + C\[4px]&= rac{(3x^2 + 4)^5}{5} + C.nd{align*}19829

Exercise: hBcsub

Find $\displaystyle \int 2\sin^3(2x)\cos(2x)\,dx$.

Solution: Let = \sin(2x)$. (Another possible value is = 2x$, but that won’t advance the problem forward — if you don’t see why, try it!) Then = 2\cos(2x)\,dx$ by the chain rule, and so the integral becomes

19829egin{align*}\int u^3\,du &= rac{u^4}{4} + C\[4px]&= rac{1}{4} \sin^4(2x) + C.nd{align*}19829

While the chain rule works for all compositions, $-sub works only in the particular situations where the derivative of the inside function is present on the outside. Unfortunately, if this isn’t the case, there’s usually not a way to get around it.

Example: hBcsub doesn't always work

Find $\displaystyle \int 6x^2(3x^2 + 4)^4\,dx$.

If we try to set = 3x^2 + 4$ again, we get that = 6x\,dx$, as before. The integral then becomes

19829egin{align*}\int xu^4\,du,nd{align*}19829

and that’s a dead end, because the integral contains more than just $. The only way to possibly make it work is to solve for $ in terms of $, which gives

and this is significantly worse than what we started with. Unfortunately, there is simply no way to make $-sub helpful on this problem. Eventually, we’ll talk about a different strategy to handle this kind of integral, but that’s still some time away.

Solution: let = 4t^3 + 3$. Then = 12t^2\,dt$, but we have t^2\,dt$, so we solve for t^2\,dt$ to get t^2\,dt = 6\,du$. Now we have

19829egin{align*}\int 2t^2 e^{4t^3 + 3}\,dt &= \int 6 e^u\,du\[4px]&= 6e^u + C\[4px]&= 6e^{4t^3 + 3} + C.nd{align*}19829

One common type of integral involves $\sin^2(x)$ or $\cos^2(x)$ (remember that those are just more convenient ways of writing ^2$ and ^2$). For these, we use a modified version of the half-angle formulas:

Proposition: the half-angle formulas

For any angle $ heta$,

1. $\displaystyle \sin^2( heta) = rac{1 - \cos(2 heta)}{2}$.

2. $\displaystyle \cos^2( heta) = rac{1 + \cos(2 heta)}{2}$.

Together, these let us integrate expressions involving $\cos^2(x)$ and $\sin^2(x)$.

Example: the half-angle formulas

Find $\displaystyle \int 3\cos^2(x)\,dx$.

First, we need to apply the half-angle formula:

19829egin{align*}\int 3\cos^2(x)\,dx &= 3\int \cos^2(x)\,dx\[4px]&= 3\int rac{1 + \cos(2x)}{2}\,dx\[4px]&= 3\int rac{1}{2} + rac{1}{2}\cos(2x)\,dx\[4px]&= rac{3}{2}x + C + 3\int rac{1}{2}\cos(2x)\,dxnd{align*}19829

To integrate $\cos(2x)$, we set = 2x$ and = 2\,dx$, so = rac{1}{2}\,du$. Now

19829egin{align*} rac{3}{2}x + C + 3\int rac{1}{2}\cos(2x)\,dx &= rac{3}{2}x + C + rac{3}{4}\int \cos(u)\,du\[4px]&= rac{3}{2}x + C + rac{3}{4}\sin(u)\[4px]&= rac{3}{2}x + rac{3}{4}\sin(2x) + C.nd{align*}19829

Although we have two integrals, we only need one $, since it can take on any numerical value.
