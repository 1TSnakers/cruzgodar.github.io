!header Section 14: Partial Fractions !text b c Covered Monday of Week 8
!begin-text-block We’re ready for our final integration technique. This one will let us handle complicated rational functions, which are just one polynomial divided by another. Compared to the first three techniques, which tell us substitutions we can make to simplify the integral, this one just splits one complicated function into many simple ones, each of which we can then integrate separately. !end-text-block !begin-def degree of a polynomial Let $P(x)$ be a polynomial. The degree of $P$, written $\deg(P)$, is the largest exponent on $x$ in $P(x)$. !end-def !begin-example degree of a polynomial $\deg(x^5 + 2x^4 - 2) = 5$. $\deg(4) = 0$. $\deg(x^3 + 2x^2 - x^3) = 2$. !end-example !begin-text-block In general, partial fractions will work on functions of the form $\frac{P(x)}{Q(x)}$, where $P$ and $Q$ are polynomials and $\deg(P) < \deg(Q)$. It will always require us to factor $Q$, so let’s take a moment to remember how polynomials can factor. Ideally, a polynomial splits into linear factors. For example, if $Q(x) = x^2 - 5x + 6$, it factors into $(x - 2)(x - 3)$. We call them linear factors because $x - 2$ and $x - 3$ are both linear functions. Sometimes these factors might be repeated: $(x-2)^2(x-3)^2$ is written with linear factors, but both $(x-2)$ and $(x-3)$ appear twice. Finally, there are some polynomials that cannot be split into linear factors. For example, $x^2 + x + 1$ cannot factor — if we try to use the quadratic formula on it, the part under the square root is $\sqrt{b^2 - 4ac} = \sqrt{1 - 4} = \sqrt{-3}$, so there are no real numbers that solve the equation. We call a factor like $x^2 + x + 1$ an irreducible quadratic, because it is quadratic and doesn’t split any further. Although the reason why is out of the scope of this class, every polynomial splits into linear and irreducible quadratic factors, possibly repeated. Partial fractions will split $\frac{P(x)}{Q(x)}$ into a handful of terms based on the factors of $Q(x)$, and we’ll handle each one with a previous integration technique. !end-text-block !begin-thm partial fractions Let $\frac{P(x)}{Q(x)}$ be a ratio of polynomials. Factor $Q$ into linear and irreducible factors, and handle one factor at a time. For a linear factor $(x - a)^n$, add $$ \frac{A}{x - a} + \frac{B}{(x - a)^2} + \frac{C}{(x - a)^3} + \cdots + \frac{D}{(x - a)^n} $$ to a running total. For a quadratic factor $(x^2 + ax + b)^n$, add $$ \frac{Ax + B}{x^2 + ax + b} + \frac{Cx + D}{(x^2 + ax + b)^2} + \frac{Ex + F}{(x^2 + ax + b)^3} + \cdots + \frac{Gx + H}{(x^2 + ax + b)^n}. $$ The capital letter variables here are just examples — make sure they’re all different. At the end, you’ll have a large sum of terms. Set it equal to $\frac{P(x)}{Q(x)}$ and multiply both sides by the factored form of $Q(x)$. Then foil out everything on the right side and set corresponding coefficients of $x$ equal to one another to solve for all the variables. !end-thm !begin-text-block There’s a lot to deal with here, but it’s not so bad in practice. Let’s start with a simple example and work our way up. !end-text-block !begin-example partial fractions Find $\int \frac{3x + 2}{x^3 - x^2 - 2x}\ dx. Let’s go down the checklist. We can’t use $u$-sub, because nothing in the denominator has a derivative matching something in the numerator. We can’t integrate by parts, because we don’t know how to integrate $\frac{1}{x^3 - x^2 - 2x}$ and differentiating it just makes it worse. Trig sub is a non-starter because there are no patterns matching it. Instead, since this is one polynomial divided by another and the degree of the top (1) is less than the degree of the bottom (3), we can use partial fractions. The first step is to factor the denominator. First we can take an $x$ out: $x(x^2 - x - 2)$. Next we can factor the remaining quadratic as $(x+1)(x-2)$, so the total polynomial becomes $x(x+1)(x-2)$. Now each of these is a linear factor (since there’s no $x^2$), so partial fractions says that the whole function decomposes as $$ \frac{3x + 2}{x(x+1)(x-2)} = \frac{A}{x} + \frac{B}{x + 1} + \frac{C}{x - 2}. $$ The next thing it tells us to do is multiply both sides by $x(x+1)(x-2)$. That gives us the equation $$ 3x + 2 = A(x+1)(x-2) + B(x)(x-2) + C(x)(x+1). $$ Now we’re supposed to foil everything out. $$ 3x + 2 = Ax^2 - Ax - 2A + Bx^2 - 2Bx + Cx^2 + Cx. $$ Now we group like terms. $$ 3x + 2 = (A + B + C)x^2 + (-A - 2B + C)x - 2A. $$ Now for these polynomials to be equal, the coefficients on $x$ must be the same, and so must the coefficients on $x^2$, and so on. That means $$ \begin{align*} 0 &= A + B + C\\ 3 &= -A - 2B + C\\ 2 &= -2A \end{align*} $$ Now we can go through and solve for $A$, $B$, and $C$. The third equation tells us that $A = -1$. Plugging that into the other two, we get $$ \begin{align*} 0 &= -1 + B + C\\ 3 &= 1 - 2B + C \end{align*} $$ Now !end-example !begin-text-buttons previous l Previous home l Home next l Next !end-text-buttons !footer
