Julia Set Explorer

The Mandelbrot set, maybe the most famous fractal ever discovered, has a family of closely related fractals — children, so to speak. Where the Mandelbrot set iterates the function \(f(z) = z^2 + c\) with \(c\) chosen as the starting \(z\)-value, Julia sets choose \(c\) beforehand and use it regardless of the starting \(z\)-value. This means that points close to \(c\) will look similar to the Mandelbrot set at that point, while the resemblance fades farther away. To emphasize that resemblance, this applet uses the Mandelbrot set to choose \(c\). Drag or hover on the Mandelbrot set to see the corresponding Julia set, and let go or click to lock it in place and apply autoexposure.
