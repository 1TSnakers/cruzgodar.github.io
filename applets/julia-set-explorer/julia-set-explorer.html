Hover over the Mandelbrot set to preview the corresponding Julia set, and click to generate a higher-resolution preview. For more information on Mandelbrot and Julia sets, have a look at this blog post.

Higher Resolution

Enter values of \(a\) and \(b\), along with an image size in pixels, to generate an image of the Julia set for \(c = a + bi\). The values are filed in automatically when a high-resolution preview is created above, but custom values can also be used. Image sizes larger than 2000 are not recommended.
