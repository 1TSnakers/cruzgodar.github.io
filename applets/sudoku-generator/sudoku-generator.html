Sudoku Generator

About six months ago, I wrote this thing over the span of a week. Not counting the website as a whole, it was the largest coding project I’d ever attempted, and I’m still amazed it works. If others on the internet have written their own generators, they’re not sharing as far as I can tell.


Maybe I should have started with this applet before building that one, but the upshot of making the more complicated one first was this only took two hours. Since it uses the same code base as the calcudoku generator, it has no way to measure difficulty, and will always create as difficult a puzzle as possible — but at least it’s quite a bit faster!
