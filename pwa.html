Add to Home Screen
cruzgodar.com is a progressive web app. That means it can be installed on your device and run like a normal application — it's like setting a bookmark, but better. Your settings are saved between sessions, and it cuts down on data usage by intelligently caching what it fetches, never downloading the same thing twice. And without a url bar to get in the way, the experience is far cleaner. If this idea sounds interesting, you can install the app by following the instructions below that apply to you.

iOS: in Safari, tap the share icon, then select Add to Home Screen in the bottom row.

Android: in Chrome, Firefox, Opera, and other browers that support it, select Add to Home Screen from the menu next to the address bar.

Desktop: In Chrome, select Install Cruz Godar from the menu next to the address bar.
